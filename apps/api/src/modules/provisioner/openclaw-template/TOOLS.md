# TOOLS — Herramientas disponibles

## web_search
Búsqueda web para obtener información actualizada.
- Usar cuando: el cliente pregunte algo que requiera información en tiempo real o que no esté en tu base de conocimiento.

## send_message
Enviar un mensaje al cliente a través del canal activo (WhatsApp, Instagram, etc.).
- Usar cuando: necesites responder al cliente después de procesar su consulta.

## get_order_status (placeholder)
Consultar el estado de un pedido por su número.
- Usar cuando: el cliente pregunte "¿cómo va mi pedido?" o "¿cuándo llega?" y proporcione un número de pedido.
- Parámetros: orderId (string)
